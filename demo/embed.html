Agent Chat Widget

A compact, embeddable LivePerson agent console — drop it into any CRM or internal tool with a single <iframe>. The live widget is on the left. Everything here shows how to add it and what it does.

Install

  1. Deploy the backend (this codebase) — it serves both the widget UI and the /api routes from one origin.
  2. Add the iframe to your page, pointing src at that backend's /widget route: 
    <iframe
  src="https://YOUR-BACKEND/widget"
  width="400" height="700"
  allow="camera; microphone; clipboard-write"
  style="border:0; border-radius:20px">
</iframe>
  3. For SSO, register <origin>/widget/callback in your LivePerson OAuth app. Password sign-in needs no setup.

camera; microphone enable the video & voice call features. SSO opens a brief popup (LP's login can't be framed); password login works in-frame.

User guide

  • Sign in — pick SSO (popup) or Password, enter your LP Account ID.
  • Switch conversations — click the chat icon in the rail, or any avatar below it. Hover an avatar to preview the last messages.
  • Reply — type in the composer. Enter sends; Shift+Enter for a new line. Attach files with the paperclip; use the sparkle menu for AI rewrite/translate.
  • Canned responses — type a hotkey prefix then press → to pick by suffix, or open the Canned tab in the panel.
  • Details panel — the person icon (top right) slides out Customer / Notes / Canned / Forms / Pages / Live.
  • Conversation actions — the ⋮ menu (top right): cobrowse, video, voice, send secure form, transfer, private message, back-to-queue, end.
  • Availability — your avatar (bottom of the rail): set Online / Away / Back soon, view skills, send all back to queue, switch theme, sign out.
